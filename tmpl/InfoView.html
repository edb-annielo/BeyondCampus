問：我是否需要付費，才能使用「移動學堂」？
答：不用，絕對免費。「移動學堂」是由教育局研發製作的電子教學工具，當然是免費使用。
問：我可否使用「移動學堂」作商業用途？
答：不可以。「移動學堂」是製作給教師在香港作學與教用途。
<% } else { %>
Q: Do I have to pay to use Beyond Campus?
A: No, absolutely free. "Beyond Campus" is an educational e-tool designed & developed by the Education Bureau, it is free to use.
Q: Can I use Beyond Campus for commerical purposes?
A: No. Beyond Campus is developed for teachers to facilitate learning and teaching in Hong Kong.

<%- dict('Account') %>

<% if(user.lang() == 1028) { %>
問：我是否必須申請一個帳戶，才能使用「移動學堂」？
答：不一定。如果你只是參加一個旅程（例如：學生），你不需要申請帳戶；但是，如果你想建立或修改旅程（例如：教師、活動策劃人），你便需要申請帳戶。
問：我是一名學生領袖，我們會舉辦一個實地考察活動，我可以申請一個帳戶嗎？
答：可以。請使用你在學校的電郵地址進行註冊。（即是你的學生電郵地址）
問：我已經成功申請了一個帳號。為什麼我不能登入？
答：當完成登記時，我們會寄出啟動帳戶的步驟到你的電郵。你必須先啟動您的帳戶才能登入。以下為常見的帳戶登記問題：
  • 電郵地址不存在，例如： 電郵地址打字錯誤、 虛假的電郵地址、 E-services的電郵地址誤寫為 "@eservice"（正確是 "@eservices.hkedcity.net"）
  • 不能接收「啟動帳戶」郵件，例如：電郵信箱已滿
  • 3天之內沒有啟動帳戶
<% } else { %>
Q: Must I sign-up an account to use Beyond Campus?
A: No. If you are only participting a trip, e.g. students, you do not need an account. However, you need to apply an account if you want to create and modify trips, e.g. teachers, event organisers.
Q: I am a student leader. We are hosting a field trip event. Can I apply an account?
A: Yes. Remember to use your email address at school to sign-up. (i.e. your student email)
Q: I have applied an account successfully. Why can't I login?
A: Once registered, an activate note will be sent to your email. You must activate your account. Belows are common sign-up problems.
  • The email address does not exist, e.g. typo in the email address, entering a fake email address, mistaking spelled E-services email as "@eservice" (should be "@eservices.hkedcity.net").
  • The activition email cannot be received, e.g. the mailbox is full.
  • The account is not activated within 3 days.
<% } %>

<% if(user.lang() == 1028) { %>
問：在各裝置/平台 / 瀏覽器上使用「移動學堂」時，是否有任何限制？
答：以下為已知限制。
  • Windows XP：
    本網站不支援Internet Explorer 8或以下，而Windows XP不能升級至 IE 9，請改用其他瀏覽器，例如：Firefox、Chrome或Safari
  • 「照相 / 選擇圖片」的任務：
    • Internet Explorer 9：不支援（請升級至 IE 10 或改用其他瀏覽器）
    • 桌上電腦 / 筆記簿電腦 (Windows / Mac)：不支援照相，但可選擇圖片
    • iOS 5：不支援（請升級至 iOS 6）
    • Android：如果電量不足，有機會作答失敗
  • 定位功能：
    • Android 的 Chrome：不能使用任何定位功能，直至Google修正其錯誤
    • 桌上電腦 / 筆記簿電腦 (Windows / Mac)：需開啟無線網路(wifi)
    • 移動裝置 (iOS / Android)：需啟用裝置的定位服務，詳見「使用定位系統 (GPS)」
  • 下載功能：
    • Android：不支援
    • iOS：不支援
    • Internet Explorer：不支援
<% } else { %>
Q: Is there any limitation when using Beyond Campus in various devices or browsers?
A: Below are known limitations.
  • Windows XP:
    This site does not support Internet Explorer 8 or below, but Windows XP cannot upgrade to IE 9. Please switch to other browsers, e.g. Firefox, Chrome or Safari
  • "Camera / Select Image" tasks:
    • Internet Explorer 9: Not supported (Please upgrade to IE 10 or switch browsers)
    • Desktop / Laptop (Windows / Mac): No camera functions, but can select image
    • iOS 5: Not supported (Please upgrade to iOS 6)
    • Android: If the battery is low, it may fail.
  • Location Service (GPS):
    • Android Chrome: Any GPS function is unavailable until Google fixes its bug.
    • Desktop / Laptop (Windows / Mac): Wifi connection must be enabled.
    • Mobile Devices (iOS / Android): "Location service" must be enabled, please refer to "Use of GPS"
  • Download Functions:
    • Android: Not supported
    • iOS: Not supported
    • Internet Explorer: Not supported
