To insert an image:
  • Upload the image file (e.g. web album, personal web space). In the editor, click 'Insert image' and input the image's URL.
To insert a file as a link (e.g. PDF, document, video, etc.):
  • Upload the file (e.g. web storage, personal web space). In the editor, click 'Insert link' , input the file's URL.

Examples of hosting services:
  • Image: Photobucket, ImageShack, Instagram, Flickr, Imgur, Picasa, etc.
  • Video: YouTube, Tudou, etc
  • File: Box, Dropbox (in "public" folder), SkyDrive, Google Drive (set as public), etc.
